### 冒泡排序
```
	public void bubbleSort(int[] a) {
	    for(int i=0; i<a.length-1; i++)
	      for(int j=0; j<a.length-1-i; j++){
	          if(a[j] < a[j+1]){
	            int temp = a[j];
	            a[j] = a[j+1];
	            a[j+1] = temp;
	          }
	      }
	}
```

### 选择排序
```
	public void selectSort(int[] a) {
		for (int i=0; i<a.length-1; i++) {
			int k = i;
			for (int j=i+1; j<a.length; j++) {
				if (a[j]<a[k]) {
					k=j;
				}
			}
			if (k != i) {
				int temp = a[i];
				a[i] = a[k];
				a[k] = temp;
			}
		}
	}
```
